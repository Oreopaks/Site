# РегионДорСтрой — сайт

Лендинг дорожно-строительной компании ООО «РегионДорСтрой» (г. Орёл): дорожное
строительство, благоустройство, аренда спецтехники, поставка сыпучих материалов.

## Что в репозитории

- `index.html` — собранный production-сайт (standalone, без зависимостей кроме Google Fonts). Открывается в любом браузере, готов к деплою.
- `assets/logo-opt.png` — логотип компании (оптимизированный).
- `AUDIT.md` — экспертный аудит сайта (UX/UI, конверсия, SEO, доступность, производительность) с приоритетами.

## Источник дизайна

Дизайн ведётся в **Claude Design**, проект `РегионДорСтрой — дизайн сайта`
(id `de7e6e79-40b1-439a-ad97-b17122492ed1`, файл `РегионДорСтрой.dc.html`).

`index.html` скомпилирован из этого источника: шаблоны (`sc-for` / `sc-if`)
развёрнуты в статическую разметку, интерактив (корзина, модалка, формы,
scroll-reveal) перенесён на чистый JS. Оригинальный `.dc.html` вне редактора
Claude Design не рендерится — ему нужен React-рантайм из `support.js`, который в
файле не подключён, поэтому в репозитории хранится собранная версия.

Пересобрать после правок дизайна: открыть проект в Claude Design, повторно
вытащить `РегионДорСтрой.dc.html` и пересобрать `index.html`.

## Деплой

Любой статический хостинг: GitHub Pages, Netlify, Vercel, обычный nginx.
Корень — `index.html`.
