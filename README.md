# РегионДорСтрой — сайт

Лендинг дорожно-строительной компании ООО «РегионДорСтрой» (г. Орёл): дорожное
строительство, благоустройство, аренда спецтехники, поставка сыпучих материалов.

## Файлы

- `index.html` — production-сайт (standalone, без зависимостей кроме Google Fonts и Яндекс.Карты).
- `assets/logo-opt.png` — логотип компании (оптимизированный).
- `favicon.svg` — фавикон (жёлтый знак «Р»).
- `privacy.html` — политика конфиденциальности (152-ФЗ).
- `robots.txt`, `sitemap.xml` — для поисковиков.
- `telegram-worker.js` — Cloudflare Worker для отправки заявок в Telegram.
- `AUDIT.md` — экспертный аудит сайта.

## Источник дизайна

Дизайн ведётся в **Claude Design**, проект `РегионДорСтрой — дизайн сайта`
(id `de7e6e79-40b1-439a-ad97-b17122492ed1`). `index.html` компилируется из него:
шаблоны развёрнуты в статику, интерактив перенесён на vanilla JS. Поверх дизайна
в `index.html` добавлен **production-слой** (SEO, формы, аналитика) — он живёт в
репозитории; при пересборке из Claude Design его нужно переносить заново.

## ⚙️ Что настроить перед запуском (TODO)

1. **Домен** — заменить `YOUR-DOMAIN` на реальный домен в: `index.html`
   (canonical, og:*, JSON-LD), `robots.txt`, `sitemap.xml`.
2. **Telegram-заявки** — задеплоить `telegram-worker.js` на Cloudflare
   (инструкция в шапке файла), затем вписать URL воркера в `index.html` →
   `CFG.tgEndpoint`. До этого формы шлют заявку в **WhatsApp** (fallback, лиды не
   теряются).
3. **Реквизиты** — вписать ИНН/ОГРН/юр.адрес в `privacy.html` и подвал
   `index.html` (плейсхолдеры `[УКАЗАТЬ]`).
4. **Реальные фото** — заменить плейсхолдеры техники/объектов.
5. **Аналитика** — вставить счётчик Яндекс.Метрики.

## Деплой

Любой статический хостинг: GitHub Pages, Netlify, Vercel, nginx. Корень —
`index.html`. После деплоя прогнать PageSpeed Insights по публичному URL.
